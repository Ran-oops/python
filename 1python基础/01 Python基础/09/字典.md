#字典
以键值对方式存在的无序数据的组合就是字典

#字典的操作

##创建字典

1.创建空字典

	变量 = {} 或者 变量 = dict()

2.创建多个元素的字典：

	方法1：

		变量 = {键:值,键:值....}

	方法2：

		变量 = dict({键:值,键:值....})

	方法3：

		变量 = dict(键=值,键=值...)

		注意：该方式键作为形参名使用不可以添加引号，必须符合变量规则

	方法4：

		变量 = dict([(键,值),(键,值)...])
		变量 = dict([[键,值],[键,值]...])
		变量 = dict(((键,值),(键,值)...))

	方法5：

		变量 = dict(zip((键，键...),(值,值...)))

	
##字典的基本操作

访问字典的元素：

	变量[键]

添加字典的元素

	变量[新键] = 值

修改字典元素

	变量[键] = 新值

删除字典元素

	del 变量[键]
